Introduction¶

In this project, we will investigate a dataset from TMDB containing information on over 10,000 movies. The dataset contains a number of attributes about each movie such as: title, cast, director, runtime, etc.

Throughout the investigation, we will be exploring the data to answer questions about one of America's most iconic actors, Nicolas Cage.

Questions:¶

1) What is the average revenue of a Nicolas Cage movie?

2) What is the average runtime for a Nicolas Cage movie in each year?

3) What are Nicolas Cage’s five highest and lowest grossing movies?

4) What is the probability of choosing a random new movie of a specific genre, featuring Nicolas Cage, in each year?

### Function that searches for number of movies in each genre for a chosen year
### and returns a list of values in alphabetical order

def fill_genre(year):
    
    year_values = []
    genre_list = ['Action', 'Adventure', 'Animation', 'Comedy', 'Crime',
       'Documentary', 'Drama', 'Family', 'Fantasy', 'Foreign', 'History',
       'Horror', 'Music', 'Mystery', 'Romance', 'Science Fiction',
       'TV Movie', 'Thriller', 'War', 'Western']
    
    # Iterate over genre_list and search for movies in given year and genre
    # then count them and append them to the year_values list
    for i in range(len(genre_list)):    
        year_values.append(df[(df.release_year == year) & 
                              (df.genres.str.contains(genre_list[i]))].original_title.count())
    
    return year_values
